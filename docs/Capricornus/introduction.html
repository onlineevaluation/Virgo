          # 魔羯座 Capricornus

          # 魔羯座是什么

          魔羯座是判题系统的核心，作为一个 c语言 测评机的存在。采用 GoLang 编写，生产动态库 (so,dll) 供其他程序调用。

          # 完成进度

          目前可以在 windows 上进行代码的编译和运行。windows 平台完善后会移植到 linux 和 macOS 上。

          # 已知BUG

          对错误的 json 数据不会进行中断，会继续执行并且最终结果为全部通过。

          # 开源地址

          https://github.com/youngxhui/Capricornus
